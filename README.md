# Telegram-бот агентства недвижимости

Telegram-бот на Python для демонстрации объектов недвижимости и приёма заявок от клиентов.

## Возможности

- каталог объектов недвижимости
- просмотр фотографий
- переключение между объектами
- оформление заявки
- отправка заявки администратору
- сохранение заявок в Excel

## Используемые технологии

- Python
- python-telegram-bot
- openpyxl
- Git
- GitHub

## Запуск

python3 bot.py
